1. Project Directory Structure
==============================================================
A project's root directory contains the following subdirectories:

Main - contains the most important project files (changes during project development and run-time)
Settings - contains project settings files (changes during project development and run-time)
History - contains historical files, e.g. historical data, alarms, etc. (only changes during project run-time)
ThinClients - contains files needed for running the project in Thin clients (generated during project development with Reliance Design)

3. Main\System Directory File Structure
==============================================================
Files in the Main\System directory are tables in the RDT format. For each table, its columns are listed.
Each column is listed on a separate line in the form Index;Data type;Name;Description.
Index is the column's order in the table.
Only some columns have names (for most object types, only columns with index 0 through 10 have names).
Column descriptions will be added gradually.
